# Knowledge Base

The agent writes here over time. Phase-2 jobs produce:

- `learnings/` — weekly markdown lessons distilled from recent trade outcomes
- `skills/` — monthly playbooks synthesized from recurring learnings
- `fundamentals/` — reference notes on companies and sectors (manual + auto)
- `technicals/` — chart patterns and indicator playbooks (manual + auto)

The committee may consult these files during analysis to inform their reasoning.
